Dawn

Un jeu minimaliste, ou pas...

Dans le cadre de ma spécialisation à l’institut IFOCOP, un exercice permettant de tester nos compétences en javascript et par extension en front nous a été donnée. Aucun framework n’était permis, seulement un code dit «Vanilla» !

Un véritable challenge pour le néophyte que j’étais, aprés deux semaines de travail "Dawn" est née .

S’inspirant d’un populaire jeu en ligne «Agar.io» ou encore "Cosmos", l’idée était pour moi de représenter un parcours d’apprentissage, la soif de connaissance très rapidement suivie de la fatigue intense qu’elle peut générer !

Pour les curieux le code source est sur mon github.

Amusez-vous bien !
